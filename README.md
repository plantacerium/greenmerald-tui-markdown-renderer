# Greenmerald TUI Mardown Renderer

![Greenmerald](./assets/greenmerald.png)
Rust CLI - Graphical Interface - To Explore Folders - Unfold Folder/s - View Files Information -  View Paths In Panels - Render Markdown FIles.

## Run Greenmerald Markdown Renderer:
```bash
cargo run
```

Commands

* q = quit program
* j = scroll up
* k = scroll down
* e = unfold all folders in current path
* t = unfold selected folder
* h = back to previous directory
* l = enter directory
* Ctrl+j = scroll up  mardown file
* Ctrl+k = sroll down markdown file
